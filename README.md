# 個人自我介紹網站

這是一個使用 GitHub Pages 建立的個人自我介紹網站。

## 如何部署

1. 將所有檔案上傳到你的 GitHub repository (`mbolruael.github.io`)
2. 在 GitHub 上，進入 Settings > Pages
3. 選擇 Source 為 "Deploy from a branch"
4. 選擇 main/master 分支和 / (root) 目錄
5. 點擊 Save
6. 等待幾分鐘後，你的網站就會在 `https://mbolruael.github.io` 上線了！

## 檔案說明

- `index.html` - 主要的自我介紹頁面

## 自訂內容

你可以編輯 `index.html` 檔案來修改：
- 個人介紹文字
- 技能標籤
- 興趣愛好
- 聯絡方式

## 測試

在上傳到 GitHub 之前，你可以直接在瀏覽器中打開 `index.html` 檔案來預覽效果。
